v1.0.2: 圆形识别_专注模式与权重降维
本版本在"快慢分流"与"高价值判断"基础上，对 UI 交互与价值评定体系进行了全面重构。系统引入专注模式开关、物理权重多维加权以及启动校准守卫，彻底剥离人工干预痕迹，回归纯视觉驱动的自动化运转。
核心功能亮点：
专注模式与沉浸式 UI
顶部居中 [ + ] 按钮，用户可一键切换专注模式。
专注模式下，全部底层白框、绿圈、蓝圈均被抹除，仅保留全场最高价值目标。该目标以红色十字准星精准标注，并以四角折线虚框（类似摄像机取景框）围绕，视觉极度纯净。
非专注模式下，不渲染高价值目标的红色十字与虚框，屏幕仅保留快慢车道的原始检测标记，便于多目标调试。
权重评分体系重构
系统从单一维度升级为四重乘区综合评定：
基础面积分（r2 x 1.2）：大体积目标天然优先。
空间纵深加权（Y 轴 1.0~1.5）：底部目标离镜头更近，获得动态加分。
画面中心加权（X 轴 1.0~1.15）：越靠近画面中心加分越高，但远低于 Y 轴权重。
迟滞防抖（x1.05）：上一帧锁定目标附近微弱优势，防止闪烁但绝不阻碍切换。
启动校准守卫
系统新增 _calibrated 状态机。未完成至少一次触摸取色前，视觉检测循环彻底挂起，屏幕仅播放原始摄像头流，不显示任何检测标记。
色彩语义重置
残缺隔离层（触边目标）：由紫色降级为纯白色标注。
拓扑拆解层（慢车道粘连目标）：改为蓝色圆框配合蓝色十字，统一视觉语言。
边缘目标高价值隔离
触边目标不进入候选池参与高价值评比，防火墙仅标注警告，不干扰完整目标评分。
按钮 UI 简洁化
取阈值与二值化按钮去除边框仅保留文字。专注按钮灰底填充的"下陷"视觉反馈标示开关状态。
