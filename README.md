# 嵌入式机器视觉方案合集

基于 **MaixCam / OpenMV / K230** 平台的竞赛级视觉算法仓库。

## 圆形识别

基于快慢车道分流的抗粘连圆形目标识别。三维联合校验 + 距离变换地形分割 + 多维权重评分。

| 版本 | 核心能力 |
|---|---|
| [v1.0.0 — 单圆，场外，粘连分流](./圆形识别/v1.0.0/) | 快慢分流 + 边界防火墙 + 距离变换分割 |
| [v1.0.1 — 高价值判断](./圆形识别/v1.0.1/) | 上述全部 + 权重评分 + 迟滞锁定 + 幻影拦截 |
| [v1.0.2 — 专注模式与权重降维](./圆形识别/v1.0.2/) | 上述全部 + 启动校准守卫 + 专注模式 + X/Y双轴加权 + 触边隔离 |

### 快速开始

1. 将对应版本的 main.py 和 gui.py 拷贝到 MaixCam
2. 按取阈值按钮点击目标球体自动提取 LAB 颜色范围
3. 按顶部 [+] 切换专注模式，画面仅保留最高价值目标的红框准星

## 黑白柱识别

基于双阈值分屏 + 线性回归的柱体巡线检测。RGB-LAB 动态取色 + 形态过滤 + ROI 内最小二乘中轴线拟合。

| 版本 | 核心能力 |
|---|---|
| [v1.0.0 — 双色巡线](./黑白柱识别/v1.0.0/) | 左/右独立取色 + 分屏二值化 + blob.code() 区分 + 线性回归拟合 + 按钮防抖 |

### 快速开始

1. 将 main.py 和 gui.py 拷贝到 MaixCam
2. 按取阈值进入取色模式，点左半屏取色1(黑柱)，点右半屏取色2(白柱)
3. 再次按取阈值退出，自动拟合巡线
