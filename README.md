# 嵌入式机器视觉方案合集

基于 **MaixCam / OpenMV / K230** 平台的竞赛级视觉算法仓库。

## 圆形识别

基于快慢车道分流的抗粘连圆形目标识别。三维联合校验 + 距离变换地形分割 + 多维权重评分。

| 版本 | 核心能力 |
|---|---|
| [v1.0.0 — 单圆，场外，粘连分流](./圆形识别/v1.0.0/) | 快慢分流 + 边界防火墙 + 距离变换分割 |
| [v1.0.1 — 高价值判断](./圆形识别/v1.0.1/) | 上述全部 + 权重评分 + 迟滞锁定 + 幻影拦截 |
| [v1.0.2 — 专注模式与权重降维](./圆形识别/v1.0.2/) | 上述全部 + 启动校准 + 专注模式 + X/Y双轴加权 |

### 快速开始

1. 将对应版本的 main.py 和 gui.py 拷贝到 MaixCam
2. 按屏幕"取阈值" -> 点击画面中的目标球体 -> 自动提取 LAB 颜色范围
3. 按顶部 [ + ] 切换专注模式

### 关键参数

| 参数 | 默认值 | 含义 |
|---|---|---|
| FILL_RATE_MIN | 0.65 | 充盈率下限 (正圆~0.785) |
| WATER_RATIO | 0.60 | 距离变换水面切割比例 |
| TRACKING_RADIUS_SQ | 2500 | 迟滞锁定圈 (50^2 px^2) |
| L_MARGIN | 30 | LAB 明度容差 |
